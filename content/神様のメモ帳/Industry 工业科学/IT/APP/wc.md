## wc

```bash
wc command
Usage:
    wc [-lwc] <file01> [file02] ...
    # <行数>    <单词数>    <字节数>    file01
    # <行数>    <单词数>    <字节数>    file02
    # <行数>    <单词数>    <字节数>    总用量

Options:
    -l --lines    只显示行数
    -w --words    只显示词数（空格为分隔符）
    -c --bytes --chars    只显示字节数

```
