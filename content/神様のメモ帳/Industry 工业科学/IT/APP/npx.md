```bash
npx command
Installation:
    * no needed to install manually (node >= 5.2)

Description:
    * npx is one-off runner of package.
    * npx 临时运行指定包，无需安装，用完即删.
    * 远程代码必须是一个模块，即必须包含 `package.json` 和入口脚本.

Usage:
    npx exec [options] <package> [args]

Options:
    -h --help

Examples:
    npx github:piuccio/cowsay hello
    npx whistle


```
