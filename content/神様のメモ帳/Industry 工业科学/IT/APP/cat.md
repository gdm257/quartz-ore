## cat

```bash
cat command
Usage:
    cat [options] [files]

Options:
    --help
    -s --squeeze-blank    压缩连续的空行到一行
    -n --number 1    对所有行添加行号，行号从 1 开始

Description:
    * tac 是反转的 cat.
    * 不指定文件则从 stdin (fd=0) 读取.

```
