## glances

```bash
glances command
Description:
    * glances 对标 top/htop.
    * glances 用法更灵活些，但性能相对更差，小鸡不推荐使用.

Examples:
    glances -w

Options:
    -w
        * 默认监听 0.0.0.0:61208.

Installation:
    * pip install glances
    - pip install bottle    # for web server mode



```
