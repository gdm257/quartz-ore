- Objects
    - Actor
        - Actor 一般是一个小人
        - Actor 用箭头指向多个 Usecases，表示 User 可以进行多种操作
    - Usecase
        - 一个椭圆，写有用例名及其他信息
        - 一般是动词词组
        - 如果将 Actor 看成 Class，那么 Usecase 就是 Method。但这 Method 并非代码级别，而是产品经理口中的 Feature
    - Use package
        - 将多个 Usecases 框起来打包为一个 Use package
