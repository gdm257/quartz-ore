## redis-server

```bash
redis-server command
Installation:
    * Windows: scoop install -g redis

Usage:
    redis-server [/path/to/redis.conf] [options]
    redis-server -
        * read config from stdin

Options:
    -h --help
    -v --version

    --port 6379
    --sentinel
        * sentinel mode

```
