- Description
    - Advanced Tables 是一个 [[Obsidian]] 插件
    - Advanced Tables 优化了 Markdown 写表格的体验
    - 支持 `|` 创建表格
    - 支持 `Tab` 光标移动到 右侧单元格
    - 支持 `Shift + Tab` 光标移动到 右侧单元格
    - 支持 `Enter` 光标移动到 下方单元格
    - 支持 `Shift + Enter` 光标移动到 上方单元格
    - 支持 Format Table
    - 支持 Move Row
    - 支持 Move Column
    - 支持 Insert Row
    - 支持 Insert Column
    - 支持 Delete Row
    - 支持 Delete Column
    - 支持 左对齐 Column
    - 支持 右对齐 Column
    - 支持 居中 Column
    - 支持 Sort by Alpha
    - 支持 Export as CSV
