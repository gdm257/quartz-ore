- Idea
    - 用法很简单。第一，使用预定义的 class name。第二，使用固定格式的 HTML
    - 美化标准 HTML（虽然还是不好看）
    - 用预定义的 class name 代替手写 CSS。甚至是代替一些简单的 JS（这在 bootstrap 3 中被称为插件）
    - 用标准 HTML 扩展出新的「语义组件」。这里的「语义」跟 [[Markdown]] 的「语义」是同一个东西，例如「表格」是一种「语义」，「一级标题」是一种「语义」。这是种广义的「组件」
